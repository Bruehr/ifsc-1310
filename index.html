“Thoughts on Web Accessibility”

Advancements have in Web Accesibility have made interacting and contributing to the web easier for groups of people who would have had trouble otherwise. Web Accessibility has helped people with disabilites, such a auditory and visual problems, have a better experience on the web. It also helps people without disabilites who might be using their mobile phone or smart TV to access the web. Whether your disabled or not Web Accesibility can make a better experience for all on the web.

What are some things you can to do to make the Web more Accessible
  • You can add images with Alt Text
  • You can allow users enlarged font sizes
  • Contrast Sensitivity
  • You can add Keyboard Navigation
  • Make Video and Multimedia Accessible
  • Descriptive URLs
  • ARIA rules
  • Avoid Using Placeholder Text in Forms
  • Minimize Use of Tables
